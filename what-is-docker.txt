What is Docker?
 Docker is a containerization tool that packages an application along with all its dependencies 
 This ensures that the application runs the same way on any system


Features of Docker 

1. Lghtweight 
 
Docker containers use fewer system resources then virtualmachines 

2. portability

Application start run on different operating system without modification 

3. Fast Deployment 

Containers start quickly and improve deployment speed 

4. Scalability 

Docker makes it easy to scale application based on demand 

5. Isolation 

Each contanier runs independently form other containers



Common Docker Commands 

1. dockr --version 
 
Displays the installed Docker versions 
Exampele:
docker --version

2. docker pull 
Downlods an images form Docker Hub 
Example:
docker pull nginx 

3. docker images 
 
Lists all available Docker images 
Example:
docker images 

4. docker run 
Runs a Docker container 
Example:
docker run nginx 

5. Docker ps 

Dispalys running containers 
Example:
docker ps 

6. docker stop

Stops a running container 
Example:
docker stopcontainer_id 

7. docker rm 

Removes a container
Example:
docker rm container_id 

8. docker rmi

Removes a Docker images 
Example:
docker rmi nginx
